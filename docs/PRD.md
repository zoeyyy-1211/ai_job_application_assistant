# PRD：AI 秋招投递决策助手

## 1. 背景

秋招过程中，学生需要同时处理大量岗位 JD、简历修改和投递进度记录。人工判断岗位匹配度和针对 JD 修改简历耗时较高，也不利于后续复盘。

## 2. 目标用户

- 2027 届准备秋招或实习的硕士/本科生。
- 目标岗位包括 AI 产品、产品经理、项目管理、数据分析、运营等。
- 希望通过 AI 工具提高求职效率，并将项目作为作品集展示。

## 3. MVP 目标

第一阶段先实现本地可运行 Demo：

- 上传简历。
- 粘贴 JD。
- 规则版岗位匹配分析。
- 保存投递记录。
- 管理投递状态。
- 展示基础数据看板。

## 4. 核心流程

1. 用户上传简历。
2. 用户粘贴岗位 JD 并填写岗位信息。
3. 系统解析简历文本。
4. 系统使用规则匹配生成分析结果。
5. 用户查看匹配度、优势、缺口和建议。
6. 系统保存投递记录。
7. 用户在投递管理页更新状态。
8. 用户在数据看板页复盘投递情况。

## 5. 第一阶段验收标准

- Streamlit 应用可以启动。
- 支持 PDF、DOCX、TXT 简历解析。
- 无 API Key 时可以完成规则匹配分析。
- SQLite 可以保存投递记录。
- 投递记录可以筛选、更新和删除。
- 数据看板可以展示状态和岗位类型统计。
