# 作品集简历项目描述

## AI 产品实习版本

- 独立设计并 vibe coding 实现「AI 秋招投递决策助手」，围绕求职者 JD 阅读、岗位匹配、简历优化和投递复盘痛点，搭建本地 Streamlit 产品 Demo。
- 设计 OpenAI-compatible 结构化分析链路，通过 Prompt 约束模型只基于原始简历和 JD 输出匹配度、证据、缺口、建议和风险，降低 AI 简历造假风险。
- 将 LLM 能力与规则 fallback 结合，在无 API Key 或模型异常时仍可完成岗位分析，提升 Demo 稳定性和产品可用性。

## 产品经理实习版本

- 从秋招投递流程出发，拆解“上传简历 - 粘贴 JD - 匹配分析 - 定制简历 - 记录投递 - 看板复盘”的核心用户路径，并沉淀 PRD 和技术设计文档。
- 设计投递记录管理与数据看板，支持状态、岗位类型、匹配度筛选，以及 CSV 导出，帮助用户复盘高匹配岗位方向和后续跟进行动。
- 以 MVP 方式逐步迭代产品，第一版完成规则匹配和记录管理，第二版补充 LLM 分析、简历导出和作品集文档，形成可讲述的产品迭代路径。

## 数据产品 / 运营分析版本

- 搭建求职投递数据闭环，将公司、岗位、平台、状态、匹配度、JD 文本和后续行动沉淀到 SQLite 数据库，支持后续分析复盘。
- 设计投递看板，展示总投递数、平均匹配度、高匹配岗位数、待跟进数量、岗位类型分布、状态分布、匹配度分布和投递趋势。
- 基于 JD 关键词抽取统计能力要求 Top 20，辅助识别目标岗位常见能力需求，为简历优化和投递策略提供数据依据。
